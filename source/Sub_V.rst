Sub function
============

Ad hoc polymorphism 'sub' function
            
=============== ========== ==============================
**Parameters**   **Type**   **description**
*vec*            Vector     The second operand as vector
=============== ========== ==============================

Returns
-------
    Vector
        The computed subbed vector

Examples
--------
>>> v=Vector(0,0,0,2,2,2)
>>> w=Vector(1,1,1,4,5,6)
>>> print(v-w)
 ( -1.0 , -2.0 , -3.0 )
