Add function
============

Ad hoc polymorphism 'add' function

=============== =========== ===============================
**Parameters**    **Type**    **Description**
*p*                Point      The second operand as Point
=============== =========== ===============================

Returns
-------
    Point
        The translated point.

Examples
--------
>>> a=Point(1,1,1)
>>> b=Point(2,2,2)
>>> print(a+b)
 ( 3.0 , 3.0 , 3.0 )
printed
