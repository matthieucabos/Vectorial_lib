Add function
============

Ad hoc polymorphism 'add' function
            
=============== ========== ==============================
**Parameters**   **Type**   **description**
*vec*            Vector     The second operand as vector
=============== ========== ==============================

Returns
-------
    Vector
        The computed added vector

Examples
--------
>>> v=Vector(0,0,0,2,2,2)
>>> w=Vector(1,1,1,4,5,6)
>>> print(v+w)
 ( 5.0 , 6.0 , 7.0 )
