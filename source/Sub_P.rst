Sub function
============

Ad hoc polymorphism 'subb' function

=============== =========== ===============================
**Parameters**    **Type**    **Description**
*p*                Point      The second operand as Point
=============== =========== ===============================

Returns
-------
    Point
        The translated point.

Examples
--------
>>> a=Point(1,1,1)
>>> b=Point(2,2,2)
>>> print(a-b)
 ( -1.0 , -1.0 , -1.0 )
printed
