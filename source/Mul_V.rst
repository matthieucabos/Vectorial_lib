Mul function
============

Ad hoc polymorphism 'mul' function
            
=============== ========== ==============================
**Parameters**   **Type**   **description**
*vec*            Vector     The second operand as vector
=============== ========== ==============================

Returns
-------
    Vector
        The computed multiplied vector

Examples
--------
>>> v=Vector(0,0,0,2,2,2)
>>> print(v*10)
 ( 20.0 , 20.0 , 20.0 )
printed
